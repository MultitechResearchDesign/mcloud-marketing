# mCloud Marketing Website

Marketing website for **mCloud** — MultiTech's IoT device management platform.

## Tech Stack

- **React 19** + **Vite**
- **Tailwind CSS v4** (via `@tailwindcss/vite`)
- **Lucide React** for icons
- **Inter** font (Google Fonts)

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output goes to `dist/`.

## Structure

| Component | Section |
|---|---|
| `Navbar` | Fixed top navigation |
| `Hero` | Main headline + dashboard mockup |
| `LogoBar` | Integration partner logos |
| `PlatformOverview` | Four platform pillars (edge, network, cloud, security) |
| `FeatureSection` | 12-feature grid |
| `Hardware` | Gateway and sensor product cards |
| `Stats` | Key metrics banner |
| `Solutions` | Pre-built solution packages |
| `Pricing` | Starter / Pro / Enterprise tiers |
| `CTA` | Call-to-action banner |
| `Footer` | Links and copyright |
